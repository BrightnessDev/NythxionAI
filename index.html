Nythxion AI - Privacy Policy

Last updated: July 4, 2026

TL;DR: We only store minimal data needed for features. No selling, no tracking, no advertising.

1. Data We Store

The bot may store the following data to provide features:

  • Discord User IDs (for leveling, moderation, and memory features)
  • Discord Server IDs (for per-server settings)
  • User-saved memory (only when using /remember)
  • Moderation logs (timeouts, bans, warnings)

2. AI Usage

When you use AI commands, your prompt is sent to an external AI provider (Groq API) to generate a response. We do not permanently store full AI conversations.

3. What We DO NOT Do

  • We do NOT sell or share your data
  • We do NOT read private messages outside bot commands
  • We do NOT track users across servers
  • We do NOT collect personal information beyond Discord IDs and bot features

4. Data Usage

All stored data is used only for bot functionality such as:

  • Leveling system progression
  • User memory (/remember feature)
  • Moderation actions and logs

5. Data Deletion

Users can request deletion of their stored data at any time by contacting the bot owner. You can also remove specific memory using the provided bot commands.

6. Data Storage

Data is stored locally on the bot’s host machine and is not shared with third parties except for AI processing requests (Groq API for /ai responses).

7. Security

We take reasonable steps to prevent abuse or unauthorized access, but no system is 100% secure.

8. Changes

This policy may be updated over time. Continued use of the bot means acceptance of the latest version.
